# Real-Time Systems Engineering (RTSE) Project 2026

Welcome to the RTSE Project 2026 repository. This project is divided into two primary phases that simulate real-time systems and inter-process communications between a 2D pseudo-3D racing simulation and external test applications utilizing TCP.

## Game Concept

**Suggested Name:** Velocity Core (or AutoDrive: TCP Dash)
**Description:** A real-time engine integrating a retro pseudo-3D racing environment with live HUD telemetry. It features a sophisticated token system, clustered time-based spawning logic, and a dynamic secondary rear-view camera rendering via TCP. This provides a robust, simulated testing ground for autonomous driving logic and real-time response mechanisms.

## Project Structure

- **[RTSE_Phase_1_V0.5_source](./RTSE_Phase_1_V0.5_source):** The primary 2D pseudo-3D engine and game core.
- **[RTSE_Phase_2_V0.5_source](./RTSE_Phase_2_V0.5_source):** Simulation test applications and external communications via Unity/Python.
